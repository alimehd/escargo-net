# L'Es-Cargo — Quebec's First Earthship Commune

> Cultivate the imagination — nurturing new ways of living, creating, and connecting.

## Stack

- React 18 + Vite
- GSAP ScrollTrigger (parallax hero)
- Vanilla CSS (no framework)

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build   # outputs to /dist
```

Connect the repo to Vercel — it auto-detects Vite. No extra configuration needed.
